# Q-send# 📦 ناقل الملفات الاحترافي (Pro File Transfer)

**المطور:** أبوالقاسم معتصم الخير  
**الترخيص:** MIT  
**الحالة:** جاهز للبناء عبر GitHub Actions

---

## 🚀 عن التطبيق
تطبيق Android احترافي لنقل الصور والفيديوهات والمستندات وجهات الاتصال **بدون فقدان الجودة** (بدون ضغط) وعبر شبكة Wi-Fi المحلية **دون الحاجة للإنترنت**.

---

## ✨ الميزات الرئيسية
- **نقل خام (Raw)**: يحافظ على الجودة الأصلية للصور والفيديوهات 4K.
- **تشفير AES**: حماية البيانات أثناء النقل.
- **استئناف النقل**: في حالة انقطاع الاتصال، يكمل من حيث توقف.
- **بدون إنترنت**: يعمل عبر Wi-Fi Direct أو نقطة الاتصال الشخصية.
- **دعم شامل**: صور، فيديوهات، مستندات (PDF، Word، Excel)، ملفات صوتية، وجهات اتصال (VCF).
- **واجهة عربية فاخرة**: بخط Cairo الأنيق وتأثيرات إضاءة احترافية.

---

## 📱 كيفية التثبيت (بدون Android Studio)
1. اذهب إلى علامة تبويب **Actions** في هذا المستودع.
2. اختر عملية **"بناء تطبيق Android"**.
3. اضغط **"Run workflow"** → اختر الفرع `main` → اضغط **"Run workflow"**.
4. انتظر دقيقتين، ثم حمّل ملف `app-debug.apk` من قسم **Artifacts**.
5. ثبّت التطبيق على هاتفك (فعّل "التثبيت من مصادر غير معروفة").
